### Claude API Configuration

To enable the live "Speak with Jesus" chat feature:

1. **Get Claude API Key**
   - Visit [Anthropic Console](https://console.anthropic.com/)
   - Generate an API key

2. **Update script.js**
   - Open `script.js`
   - Replace this line:
     ```javascript
     const CLAUDE_API_PLACEHOLDER = 'YOUR_CLAUDE_API_KEY';
     ```
   - With your actual key:
     ```javascript
     const CLAUDE_API_PLACEHOLDER = 'sk-ant-api03-...';
     ```

3. **For Production**
   - Use environment variables
   - Implement a backend proxy to protect your API key
   - Never expose the key in client-side code
